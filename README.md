# 📝 Danh Sách Việc Cần Làm (To-Do List)

Một ứng dụng to-do list đơn giản nhưng mạnh mẽ với chức năng lưu trữ cục bộ (Local Storage).

## ✨ Tính Năng

- ✅ **Thêm công việc mới** - Nhập và thêm công việc vào danh sách
- ✅ **Đánh dấu hoàn thành** - Bật/tắt công việc khi hoàn thành
- ✅ **Xóa công việc** - Xóa công việc riêng lẻ
- ✅ **Lọc công việc** - Xem tất cả, đang làm, hoặc hoàn thành
- ✅ **Thống kê** - Hiển thị tổng số công việc và công việc hoàn thành
- ✅ **Local Storage** - Lưu tự động các công việc trong trình duyệt
- ✅ **Responsive Design** - Hoạt động tốt trên mọi thiết bị

## 🚀 Cách Sử Dụng

1. **Mở file `index.html`** trong trình duyệt
2. **Nhập công việc** vào ô input và nhấn "Thêm" hoặc phím Enter
3. **Đánh dấu hoàn thành** bằng checkbox
4. **Lọc công việc** sử dụng nút Tất cả / Đang làm / Hoàn thành
5. **Xóa công việc** bằng nút Xóa
6. **Xóa tất cả hoàn thành** bằng nút "Xóa hoàn thành"

## 📁 Cấu Trúc File

```
.
├── index.html      # Giao diện chính
├── style.css       # Tệp CSS - thiết kế và kiểu dáng
├── script.js       # Tệp JavaScript - logic xử lý
└── README.md       # File hướng dẫn (file này)
```

## 💾 Local Storage

Ứng dụng tự động lưu toàn bộ công việc vào Local Storage của trình duyệt. Dữ liệu sẽ được:
- ✅ Lưu tự động mỗi khi bạn thêm, xóa hoặc cập nhật công việc
- ✅ Khôi phục tự động khi mở lại trang
- ✅ Xóa khi bạn xóa công việc
